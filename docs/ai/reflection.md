# Reflection


## Uso de la IA durante el proyecto

Durante el desarrollo utilicé varias herramientas de inteligencia artificial como ChatGPT, Claude y Cursor.

Estas herramientas me ayudaron principalmente a:

- comprender conceptos técnicos de JavaScript
- refactorizar funciones del proyecto
- generar documentación
- mejorar la estructura del código

También utilicé Cursor para analizar el código del proyecto y obtener sugerencias de mejora.


## Ventajas del uso de IA

El uso de IA aporta varias ventajas en el desarrollo de software:

- ayuda a resolver dudas técnicas rápidamente
- permite generar ejemplos de código de forma rápida
- facilita la refactorización y mejora del código
- ayuda a generar documentación de manera más eficiente

Esto permite ahorrar tiempo y mejorar la productividad.


## Limitaciones encontradas

A pesar de sus ventajas, las herramientas de IA también tienen algunas limitaciones:

- en ocasiones generan código que necesita ser revisado
- es necesario entender el código para verificar que funciona correctamente




## Conclusión personal

El uso de inteligencia artificial puede ser muy útil como apoyo al desarrollo, pero no sustituye el conocimiento del programador.

Utilizada correctamente, la IA puede convertirse en una herramienta muy potente para mejorar la productividad, aprender nuevos conceptos y desarrollar proyectos de forma más eficiente.
