TaskFlow es una aplicación sencilla de gestión de tareas pensada para practicar JavaScript moderno y buenas prácticas en frontend. Permite crear, completar/deshacer, eliminar y buscar tareas, mostrando por separado las tareas pendientes y las completadas, además de un contador de pendientes actualizado en tiempo real.

##Objetivo##: ofrecer una pequeña app tipo “to‑do list” bien estructurada, fácil de entender y extender (por ejemplo, añadiendo categorías, fechas o prioridades), ideal para uso didáctico o como base de proyectos personales.

##Tecnologías utilizadas##:

HTML5 para la estructura de la interfaz.
Tailwind CSS para el diseño visual y la maquetación responsive.
JavaScript puro (ES6+) para la lógica de negocio, manipulación del DOM y persistencia.

##Funcionalidades principales##:

*Añadir nuevas tareas desde un formulario.
*Marcar tareas como completadas y deshacer su estado.
*Eliminar tareas individuales.
*Filtrar tareas por texto mediante un buscador.
*Contador de tareas pendientes visible en el encabezado de la lista.
*Persistencia de todas las tareas usando localStorage, de modo que se mantienen entre recargas del navegador.
*Soporte de modo claro/oscuro con persistencia de la preferencia de tema.
